# Personal Portfolio Starter

This is a responsive single-page portfolio built with plain HTML, CSS, and JavaScript.

## Files

- `index.html`: Main page structure
- `styles.css`: Theme, layout, responsive styling, and animation
- `script.js`: Mobile navigation and scroll reveal logic

## Run Locally

1. Open this folder in VS Code.
2. Open `index.html` in your browser.

## Customize

- Update your name, role, and bio in `index.html`.
- Replace project cards in the `#projects` section.
- Change colors in the `:root` variables in `styles.css`.
- Add real social/profile links in the footer.
- Update the email link in the contact section.
